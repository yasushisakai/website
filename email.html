1 offlineimap で同期

offlineimap を使う。 python2 を使うあたり、アップデートされなさがやばいが、しょうがないです。 

# 入れてみる。
python2 -m pip install --user offlineimap
# 見つかるかしら？
which offlineimap
# 僕は見つからなかったので、macのmdfindを使って
mdfind offlineimap
# それでbinフォルダに入ってるものをPATHを追加する

ってのがめんどくくさかった。

gmail はこの方法で google service のログインとは別のパスワードを入れる必要あり。

offlineimap が使えてそうだったら、 ~/.offlineimaprc をとりあえず下記に設定する。 

cat ~/.offlineimaprc

[general]
accounts = Gmail

[Account Gmail]
localrepository = Gmail-Local
remoterepository = Gmail-Remote

[Repository Gmail-Local]
type = Maildir
localfolders = ~/Documents/Mail/gmail

[Repository Gmail-Remote]
type = IMAP
remotehost = imap.gmail.com
remoteport = 993
remoteuser = you@gmail.com
remotepass = password
ssl = yes
maxconnections = 1
realdelete = no
sslcacertfile=/usr/local/etc/openssl@1.1/cert.pem

ここでも罠があって、 sslcacertfile 箇所。mac だと僕の場合ここにあった。 openssl が入っていない場合は brew install openssl して　 which openssl で出てくるやつパスを参考にした。Linux 勢は /etc/ssl/certs/ca-certificates.crt といったところにあるらしいけど確認してください。

この時点で設定がうまくいっていれば、 offlineimap コマンドでとりあえずバックアップとってくれます。

2 履歴バックアップをとる

この offlineimap なるものはリモート側とローカルの sync をしているだけなので、本当の backup は 履歴をとることが 重要です。git を使いましょう。 

cd ~/Documents/Mail
git init
git add .
git commit -m "initial backup"

mac の場合はその上で、 timemachine とかでバックアップをとっていたりすると、さらにバックアップですね。 (それに頼る場合は git いらないかも?)

3 Interval で繰り返しやる

このままだと、同期はコマンド一度っきりなので、 offlineimap をなんらか回し続けないといけません。 deamon 化するのは失敗するというのを見たことがあるので、 cron job, systemctl, launchd みたいなのを使って数分に一回実行します。

mac なので、 launchd で描きます。 

<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE plist PUBLIC "-//Apple//DTD PLIST 1.0//EN" "http://www.apple.com/DTDs/PropertyList-1.0.dtd">
<plist version="1.0">
  <dict>
    <key>Label</key>
    <string>offlineimap</string>
    <key>ProgramArguments</key>
    <array>
        <string>/Users/yasushi/Library/Python/2.7/bin/offlineimap</string>
        <string>-o</string>
        <string>-u</string>
        <string>quiet</string>
    </array>
    <key>StartInterval</key>
    <integer>180</integer>
    <key>ExitTimeOut</key>
    <integer>120</integer>
    <key>StandardOutPath</key>
    <string>/Users/yasushi/.config/offlineimap/offlineimap.stdout</string>
    <key>StandardErrorPath</key>
    <string>/Users/yasushi/.config/offlineimap/offlineimap.stderr</string>
  </dict>
</plist>

launchctl load ~/LaunchAgents/offlineimap.plist

ほいで、最後は mutt でターミナル上でメールできるようにする。 

brew install mutt

この mutt というプログラムはそれはそれで沼なので、設定ファイルを育てていく感じになるかと思います。とりあえず、見える形にするために、設定したファイル。 

cat ~/.config/mutt/muttrc

## for sending mails
set realname = "Your Name"
set from =  "you@gmail.com"
set smtp_url = "smtps://ysshski@gmail.com@smtp.gmail.com:465/"
set smtp_pass = "pass"
# you need to set your sendmail program
# set sendmail="/usr/bin/esmtp"
set smtp_authenticators = 'login'

## appearance
set sort = 'threads'
set sort_aux = 'reverse-last-date-received'
set charset = "utf-8"

## for reading emails, read from local
# needs to be consistent with offlineimap
set mbox_type = Maildir
set folder = ~/Documents/Mail/gmail
# folder in which to start mutt
set spoolfile = +/INBOX/
# cache for even faster
set header_cache = ~/Documents/Mail/gmail/cache

## writing mail
set editor = "vim"
